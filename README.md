## Installation & Setup

### Prerequisites

- **Node.js** (v14+)
- **npm**
- **Wallet** (e.g., MetaMask)

### Steps

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/Web3masters/Agentify.git
   cd Agentify
   ```

2. **Install Dependencies:**

   ```bash
   npm install
   ```

3. **Configure Environment Variables:**  
   Create a local `.env` file and add:

   ```env

   OPENAI_API_KEY=your_openai_api_key
   
2. **Run Project:**

   ```bash
   npm run start
   ```
